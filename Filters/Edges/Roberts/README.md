Description
----------

Calculates the first derivative of an image and is used as an edge operator. 

Input parameters
---------------

**FilterType** - Filter type.

*Default:* 'gradient_sum'

*List of values:* 'gradient_max', 'gradient_sum', 'roberts_max'
