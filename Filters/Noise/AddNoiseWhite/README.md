Description
----------

Adds noise to the image. The noise is white noise, equally distributed in the interval [ *-Amp*, *Amp* ].

Input Parameters
----------

**Amp** - Maximum noise amplitude.

*Default value:* 60

*Suggested values:* 1.0, 2.0, 5.0, 10.0, 20.0, 40.0, 60.0, 90.0
