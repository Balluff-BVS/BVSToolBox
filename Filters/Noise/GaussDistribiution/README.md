Description
----------

Generate a Gaussian noise distribution. The parameter *Sigma* determines the noise's standard deviation.

Input Parameters
----------

**Sigma** - Standard deviation of the Gaussian noise distribution.

*Default value:* 2

*Suggested values:* 1.5, 2.0, 3.0, 5.0, 10.0
