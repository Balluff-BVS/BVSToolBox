This tool can be used to build custom tools for machine vision products made by Balluff.
This repository was designed as an open tools database which BVS (Balluff Vision Solutions) users can incorporate in their machine vision applications.
Scripts are divided into sections what makes it easier to find a proper tool. The whole repository is stored on the GitHub so everyone is welcomed to take a part in the development.
